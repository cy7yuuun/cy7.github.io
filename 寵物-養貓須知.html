貓的叫聲

貓，通常指家貓，為中型貓科動物。 根據遺傳學及考古學分析，人類養貓的紀錄可追溯至10,000年前的新月沃土地區， 古埃及人飼養貓的紀錄可追溯至1000年前，以防止老鼠吃掉榖物。現在，貓成為世界上最為廣泛的寵物之一，飼養率僅次於狗， 但同時也威脅著很多原生鳥類、齧齒類的生存，是世界百大外來入侵種之一。長期飼養的貓平均壽命為12年以上，歷史上最長壽的貓則達38歲。

新手第一次養貓該怎麼辦？需要準備哪些東西呢？可愛貓咪人人都愛， 但對沒養過貓的人來說，要真的跨出成為貓奴的第一步其實並不容易，很多人都說「養貓比養狗容易」， 因為不需要帶貓咪出去散步等等，但養貓仍有非常多需要了解的注意事項， 以下是今天要分享給各位的「新手貓奴全攻略」，馬上看下去吧！

養貓需要準備哪些東西？

  1. 床窩
    2. 貓咪一天會睡 10 多個小時，可以為他準備一個專屬的床窩，建議不要放在空氣流動太大或有太多人進出活動的地方。
  2. 貓砂 & 貓砂盆
    3. 大部分的貓咪因為天性使然，不需要特別訓練就會固定在貓砂裡頭大小便，因此別忘了準備貓砂和貓砂盆！
  3. 水碗
    4. 貓隨時都需要有一碗清水。您可以使用飲水噴泉取代一般水碗，因為經過測試，貓特別喜歡這種流動式的飲水，而且這種飲水噴泉也可以鼓勵牠們多喝水。
  4. 食物碗
    5. 建議準備一個可以放乾飼料的小型食物碗，而且不要和水碗放在一起，避免飲水變髒。另外，您也可以使用食物玩具，特別是如果您愛貓是室內貓，更應該使用這種食物玩具，因為這可以為牠們提供心理和生理層面的刺激，激發牠們的自然狩獵本能。有了這些食物玩具，您的愛貓需要一個過渡期來學習如何使用這些玩具。
  5. 飼料
    6. 新手貓奴建議一開始先購入專業的貓飼料，在有一定經驗後可自行決定是否讓貓咪改成濕食或生食。
  6. 貓跳台
    7. 對貓而言，貓跳台很好玩，而且很實用，因為貓可以抓、攀爬和安全躲藏，也由於這可以滿足貓的自然天性需求，所以對牠的身心健康也很有幫助。
  7. 外出籠
    8. 千萬別太相信自己抓貓的技術，外出籠絕對是必備的養貓用品，外出時請將貓咪放在專用外出籠 / 外出包內，以免貓咪一緊張亂跑、發生憾事。
  8. 貓指甲剪
    9. 貓的指甲會不斷生長，而家貓通常能夠磨指甲的機會較少，因此一支貓咪專用的指甲剪也是不可或缺的。

養貓需要多少花費？

以最基本的花費而言，養一隻貓咪每月的伙食費和貓砂費用， 最少約需新台幣 2～3000 元以上，而一開始購入養貓所需的用品也同樣需要新台幣 2～3000 元左右， 若選擇較好的飼料、貓砂等，則費用可能會再略高一些， 若貓咪身體出問題，找獸醫治療就又是一筆額外的開銷了。

養貓前必須評估的事情

  1. 有足夠的活動空間
    2. 養貓所需的空間雖然比養狗小很多，但仍需要提供給貓咪足夠的活動空間，原則上空間越大越好、最起碼貓咪能夠有空間奔跑。
  2. 環境別太髒亂
    3. 環境亂容易找不到貓咪在哪，而環境髒的話則可能讓貓咪誤食髒東西，還可能讓貓咪因此出現皮膚、呼吸道等問題，因此保持環境衛生也是很必要的。
  3. 是否有其他寵物
    4. 有些貓咪非常親人、但無法和其他動物和平共處，若家中已有其他動物，請特別謹慎評估貓咪能否順利融入環境。
  4. 最好能有對外窗
    5. 能有對外窗是大加分！對外窗也能讓室內空氣更流通、也能讓貓咪曬到太陽，且對貓咪來說，窗外景色就像一台永遠看不膩的電視，能夠看到窗外的人事物，對貓咪而言是很開心的事情；但新手貓奴千萬要小心，對外窗的防護措施一項都不能少，以免貓咪不小心摔出窗外、發生憾事。
  5. 足夠的耐心及距離
    6. 並非所有貓咪都會馬上很親人，尤其當貓咪來到一個全新的環境，難免會緊張害怕、甚至難以接近，若貓咪一開始常常躲起來，這時候就先別打擾他了！給他時間適應，信任感逐漸建立後戒心才會慢慢放下，而這通常需要一些時間。

養貓的好處 & 壞處

  • 養貓好處
  1. 生活中多一個伴
  2. 養成你固定外出活動的習慣
  3. 療癒身心、釋放壓力
  4. 結交擁有共通話題的貓飼主朋友
  • 養貓壞處
  1. 照顧貓咪的責任
  2. 增加固定開銷（尤其是帶貓貓看醫生）
  3. 出遠門較不方便
  4. 需改變家中擺設

養貓的居家環境需注意什麼？

  • 餵食區：
    • 餵食區必須在安靜的地區，遠離貓砂盆和您自己的飲食區。建議不要在餐廳或可能會讓貓咪受到干擾的區域餵食，並在餵食的時候讓貓有足夠的空間。
  • 休息區：
    • 根據陽光和熱源的不同，貓的休息區在一天之中也會移動。就本質上而言，貓喜歡在高處睡覺、會選擇最適合牠們的地方，所以請確保牠們有足夠的空間可以選擇。
  • 清潔區：
    • 貓砂盆必須放在安靜的地方，且必須方便貓隨時進出使用、遠離食物碗以及許多人進出的地方。 注意避免為牠帶來任何壓力，例如貓砂盆如果放在洗衣機旁邊，貓可能會拒絕使用。每隻貓必須至少要有一個貓砂盆，還要一個備用貓砂盆，並建議為貓提供足夠的選擇，避免牠不願意使用貓砂盆。
  • 遊戲區：
    • 是貓最大的生活空間和休息區。這個空間必須可以讓貓盡情奔跑、躲藏或在高處休息。貓跳台是讓貓可以從事這些活動的最好輔助，且可以促進牠們的身心健康。

貓貓的情緒

  1. 放鬆安心
    2. 貓在自己熟悉的環境，身體肌肉和面部線條都很放鬆，呈現懶洋洋、瞇著眼睛的樣貌，有時喉嚨還會發出咕咕的聲音，這時很適合摸摸牠。
  2. 磨蹭
    3. 剛住進來的新貓到處磨蹭，有時也會磨蹭人類的小腿，這並不是在撒嬌，而是佔地盤的動作。貓的領地意識強，其皮膚會分泌腺體，藉由磨蹭將氣味留在物體上，熟悉的氣味讓自己安心，也宣示已佔領此地。
  3. 不開心
    4. 貓不爽時表情不會有太大變化，但耳朵會稍微往後，身體大部分呈放鬆狀態，背部和尾巴的毛不會豎起。尾巴會快速左右擺動，如果持續讓牠們不悅的話，也會出現輕咬和用前腳拍打的動作。
  4. 緊張害怕
    5. 貓在緊張害怕時瞳孔會變圓，耳朵往後或往旁邊下壓，表情變得僵硬，眼睛會注意讓牠緊張的人事物，身體壓低或趴下，尾巴捲起在兩腿間，甚至做好逃跑的準備動作。
  5. 生氣
    6. 貓生氣時瞳孔會放大，背部和尾巴的毛直豎，也就是俗稱的「炸毛」。背部會像弓一樣拱起，讓自己體型變大助長氣勢。另外叫聲會變得尖銳，露出牙齒發出嘶嘶的哈氣聲。
